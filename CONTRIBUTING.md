# Contributing to FOCUSYNC 💙

Thank you for your interest in contributing to FOCUSYNC!
FOCUSYNC is a privacy-first productivity and burnout awareness tool for developers.

## Ways to Contribute
- Improve UI/UX or accessibility
- Add new analytics visualizations
- Improve AI Coach logic or prompts
- Fix bugs or improve performance
- Improve documentation

## Getting Started
1. Fork the repository
2. Clone your fork
3. Create a new branch:
   git checkout -b feature/your-feature-name
4. Make your changes
5. Commit with clear, descriptive messages
6. Push and open a Pull Request

## Code Guidelines
- Use TypeScript best practices
- Keep components small and reusable
- Avoid introducing backend services (privacy-first principle)

## Issues
Please check existing issues before opening a new one.
Comment on an issue if you would like to work on it.

Thank you for helping make FOCUSYNC better 🚀
